# Pretina V2

Enterprise-grade e-commerce platform — MERN + React Native monorepo.

## Structure

```
pretina/
├── backend/       → Node.js + Express + MongoDB API
├── admin/         → React + Vite Admin Panel
└── app/           → React Native Customer App
```

## Tech Stack
- **Backend:** Node.js, Express, MongoDB (Mongoose), Firebase Admin SDK, AWS S3
- **Admin:** React, Vite, Redux Toolkit, MUI
- **App:** React Native (CLI), Redux Toolkit, Firebase Auth (Phone OTP)
- **Infra:** AWS Lightsail, MongoDB Atlas, AWS S3

## Phases
- Phase 1 (Days 1–10): Backend Foundation ← **CURRENT**
- Phase 2 (Days 11–20): Admin Panel
- Phase 3 (Days 21–35): Mobile App
- Phase 4 (Days 36–42): Data Migration & QA
- Phase 5 (Days 43–50): Production & Deployment
